# AWS & Terraform Interactive Learning Portal

A comprehensive, interactive HTML-based learning portal for AWS and Terraform fundamentals with complete offline support. Built to help learners master cloud infrastructure and Infrastructure as Code through structured content, interactive flashcards, and practical examples.

## 🎯 Project Overview

This is a **standalone, single-file learning application** that provides a complete interactive curriculum covering:
- **AWS Fundamentals** (Regions, IAM, EC2, VPC, S3, Route 53)
- **Terraform & Infrastructure as Code** (Workflow, State Management, Modules, Best Practices)
- **30 Topics** organized in 6 weeks
- **30 Flashcards** for active learning and review
- **No external dependencies** - runs entirely in the browser

## ✨ Features

- 📚 **30 Comprehensive Topics** - Structured AWS and Terraform content
- 🎴 **Interactive Flashcard System** - Flip-based learning cards with explanations
- 📊 **Progress Tracking** - Browser-based progress saving (localStorage)
- 🔍 **Search Functionality** - Find topics quickly
- 📐 **Diagrams & Layouts** - Visual aids for every topic
- ⚡ **Zero Dependencies** - No CDN, no build process, no installation
- 🌐 **Offline-First** - Fully functional without internet
- 📱 **Responsive Design** - Works on desktop and mobile

## 📋 Curriculum Structure

### Weeks 1-3: AWS Foundations
- Cloud models (IaaS, PaaS, SaaS)
- Global infrastructure & regions
- Identity & Access Management (IAM)
- EC2, VPC, Security Groups
- Storage & Networking

### Weeks 4-6: Terraform & Architecture
- Infrastructure as Code principles
- Terraform workflow (init, plan, apply)
- State management & backends
- Modules & reusability
- Architecture patterns & DevOps
